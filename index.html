The default sequence is of the Ebola Virus. Just press the draw-button check it out!

Play around with the settings and create something brand new.

You can find the source code here.

If you want to use something created with this, I'd absolutely LOVE(!) to hear about it. Please send me a message either on Reddit or via Mail: biegler.dan (at) gmail (dot) com

Tips:

  • If you're on mobile press the draw-button and then zoom out to see the whole image.
  • The sequence input is case insensitive.
  • You can right-click the image to "Save image as...". The white background is transparent when you save!
  • You can click on the color-boxes in the settings section to change them! After that, redraw the image with the draw-button.
  • You can find a couple of DNA sequences here.
